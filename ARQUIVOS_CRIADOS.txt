🎯 FLUXO DE USO
════════════════════════════════════════════════════════════════════════

1️⃣  INICIAR
    └─ Duplo-clique: executar_streamlit.bat
       └─ navegador abre em http://localhost:8501

2️⃣  CONFIGURAR (primeira vez)
    ├─ Edite: .env
    │  ├─ API_BASE_URL
    │  ├─ API_KEY
    │  └─ RATE_LIMIT_REQUESTS
    │
    └─ Edite: api_rastreamento.py
       ├─ Método: consultar_por_nf()
       ├─ Método: consultar_por_rastreamento()
       └─ Método: _processar_resposta_api()

3️⃣  TESTAR
    └─ Streamlit → 🏠 Início → "Testar Conexão com API"
       └─ Se OK: verde ✅
           Se erro: verifique .env

4️⃣  CRIAR MODELO
    └─ Streamlit → ⚙️ Processar → "Criar Modelo"
       └─ Gera: dados/rastreamento_modelo.xlsx

5️⃣  INCLUIR DADOS
    └─ Abra: dados/rastreamento_modelo.xlsx
       ├─ Coluna A: NF (número nota fiscal)
       ├─ Coluna B: Rastreamento (código rastreamento)
       └─ Preencha com seus dados

6️⃣  PROCESSAR
    └─ Streamlit → ⚙️ Processar → "Processar Rastreamentos"
       ├─ Backup automático
       ├─ Consulta API
       ├─ Atualiza Excel
       └─ Gera relatório

7️⃣  VISUALIZAR
    └─ Streamlit → 📊 Visualizar Dados
       ├─ Vê todos os registros
       ├─ Filtra por status
       └─ Busca por tracking

8️⃣  GERENCIAR BACKUPS
    └─ Streamlit → 💾 Backup e Restauração
       ├─ Lista backups
       ├─ Vê metadados
       └─ Restaura se necessário
