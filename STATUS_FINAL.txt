⚙️ CONFIGURAÇÃO NECESSÁRIA
══════════════════════════════════════════════════════════════════════════

1. ARQUIVO .env
   ────────────────────────────────────────────────────────────────────
   
   Edite: c:\Automação.Transportadora\.env
   
   Altere:
   ┌─────────────────────────────────────────────────────────────────┐
   │ API_BASE_URL=https://www.siterastreio.com.br/api-correios               │
   │ API_KEY=sua-chave-api                                           │
   │ API_TIMEOUT=30                                                  │
   │ API_RETRIES=3                                                   │
   │ RATE_LIMIT_REQUESTS=100                                         │
   │ RATE_LIMIT_PERIOD=60                                            │
   │ LOG_LEVEL=INFO                                                  │
   │ CAMINHO_EXCEL=dados/rastreamentos.xlsx                          │
   │ PASTA_BACKUPS=backups                                           │
   └─────────────────────────────────────────────────────────────────┘

2. API_RASTREAMENTO.PY
   ────────────────────────────────────────────────────────────────────
   
   Métodos que precisam adaptação:
   
   a) consultar_por_nf()
      └─ Altere: endpoint, parâmetros, headers conforme sua API
      
   b) consultar_por_rastreamento()
      └─ Altere: endpoint, parâmetros, headers conforme sua API
      
   c) _processar_resposta_api()
      └─ Altere: mapeamento de campos da resposta JSON
      
   Veja exemplos em: EXEMPLOS_INTEGRACAO.py

3. CONFIG.PY
   ────────────────────────────────────────────────────────────────────
   
   Atualize o dicionário MAPEAMENTO_STATUS:
   
   Padrão (8 status):
   ┌─────────────────────────────────────────────────────────────────┐
   │ "PENDENTE"          → "⏳ Pendente"                             │
   │ "EM_TRANSITO"       → "🚚 Em Trânsito"                         │
   │ "ENTREGUE"          → "✅ Entregue"                            │
   │ "ATRASO"            → "⚠️ Atraso"                              │
   │ "DEVOLVIDO"         → "↩️ Devolvido"                           │
   │ "CANCELADO"         → "❌ Cancelado"                           │
   │ "ERRO"              → "🔴 Erro"                                │
   │ "DESCONHECIDO"      → "❓ Desconhecido"                        │
   └─────────────────────────────────────────────────────────────────┘
   
   Adapte para sua API (adicione/remova conforme necessário)
